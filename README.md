# Hina Landing Page 🚀

Este projeto é uma Landing Page moderna desenvolvida com **Next.js 14**, **Tailwind CSS** e **TypeScript**.

### 🛠️ Status do Projeto: 
🚧 **Em desenvolvimento ativo** 🚧

### 🎯 O que já funciona:
- Interface responsiva com Tailwind CSS.
- Estrutura base com Next.js (App Router).
- Lógica inicial de cronômetro para treinos.

### 🏁 Próximos Passos:
- Finalização da estilização dos componentes.
- Refatoração de hooks personalizados para o timer.
- Deploy oficial na Vercel.
