## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as ferramentas mais modernas do ecossistema Web:

* **Next.js 15 (Turbopack)**: Estrutura escalável e carregamento ultra-rápido.
* **Tailwind CSS**: Estilização baseada em utilitários para um design "Dark Mode" premium.
* **TypeScript**: Tipagem estática para um código mais seguro e fácil de manter.
* **React Hooks (useState, useEffect)**: Gerenciamento de estado complexo para o cronômetro.
* **Git**: Controle de versão e resolução de conflitos em ambiente de desenvolvimento real.

## 🧠 Desafios e Aprendizados

O maior desafio técnico deste projeto não foi apenas o visual, mas a **engenharia da lógica de tempo**. Criar um cronômetro que alterna automaticamente entre estados de *Preparação*, *Round* e *Descanso* exigiu um entendimento profundo de como o React lida com ciclos de renderização e efeitos colaterais.

Além disso, o processo de desenvolvimento envolveu a resolução de conflitos de merge no Git, o que reforçou a importância de uma boa organização de arquivos e do controle de versão profissional.
